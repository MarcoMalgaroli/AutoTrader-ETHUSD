# AutoTrader-ETHUSD
AI-powered ETHUSD automated trading system with predictive models and operational integration with MetaTrader5 platforms.

# How to run
Make sure to have a MetaTade5 terminal installed in your system.

Optionally, you can create a virtual environment to manage dependencies:

```bash
python -m venv .venv
.venv\Scripts\activate
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

Then, you can run the main script to start the trading system:

```bash
python -m dashboard.app
```
